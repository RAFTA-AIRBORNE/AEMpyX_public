# PROJECT_inv_synth_readme.md

## Script: `PROJECT_inv_synth.py`

**Purpose:** AEMpyX synthetic data inversion ensemble.
Inverts a set of synthetic data realisations (generated by
`PROJECT_fwd_synth.py`) to build a posterior ensemble, then
computes summary statistics (quantiles, mean, std, skew, kurtosis,
mode) for models, data, and residuals.

## Key Parameters

| Variable | Description |
|---|---|
| `RunType` | `'TikhOpt'` / `'TikhOcc'` / `'MAP'` |
| `Nlyr` | Number of inversion layers |
| `Guess_rv` / `Guess_rs` | Prior resistivity mean and log-std |
| `PerturbPrior` | If `True`, perturb prior for each sample |
| `PertVar` | Per-sample perturbation standard deviation (log scale) |
| `PertShift` | Global shift applied to all active parameters |
| `EnsOut` | If `True`, save full ensemble; else save statistics only |
| `Percentiles` | Quantiles to compute (e.g. `[10, 20, 30, 40]`) |
